◆ getTagValue()

QString getTagValue ( const QString & response,
const QString & tagName )

Extracts the value enclosed within a specified XML tag from a given response string.

This function searches for the specified opening and closing tags within the response string and extracts the text that lies between these tags. If the tags are not found, it returns an empty string.

Parameters
responseThe input string containing the XML-like content.
tagNameThe name of the tag whose value needs to be extracted.
Returns
A QString containing the text between the specified tags, or an empty QString if the tags are not found.

◆ resetValues()

void resetValues ( )

Resets the game values to their initial state.

This function performs the following actions:

  • Sets the game_started flag to "0".
  • Clears the message string.
  • Initializes the board array with the value "-".

Note: Some variables such as connect_arduino, game_mode, and ai_strategy are commented out and not reset by this function.
